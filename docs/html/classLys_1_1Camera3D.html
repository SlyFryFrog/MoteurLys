void Lys::Camera3D::look_at ( const Position3 &  point)

Sets the camera's view matrix to look at a specified point.

This function updates the camera's view matrix to orient the camera such that it looks at the given point in 3D space. The view matrix is calculated based on the camera's current position and the specified point, while maintaining the camera's up direction as defined by worldUp.

Parameters
pointThe target position in 3D space that the camera should look at.

Example usage:

Position3 targetPoint = {1.0f, 2.0f, 3.0f};
camera.look_at(targetPoint);
