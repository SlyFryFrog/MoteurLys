{\footnotesize\ttfamily void Lily\+Pad\+::\+Camera3\+D\+::look\+\_\+at (\begin{DoxyParamCaption}\item[{const \mbox{\hyperlink{structLilyPad_1_1Vector3}{Position3}} \&}]{point }\end{DoxyParamCaption})}



Sets the camera\textquotesingle{}s view matrix to look at a specified point. 

This function updates the camera\textquotesingle{}s view matrix to orient the camera such that it looks at the given point in 3D space. The view matrix is calculated based on the camera\textquotesingle{}s current position and the specified point, while maintaining the camera\textquotesingle{}s up direction as defined by {\ttfamily world\+Up}.
